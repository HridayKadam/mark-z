# Mark Z Agency

A minimalistic, paper-themed agency website for Mark Z, featuring an academic aesthetic focused on design, video, and AI automations.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
